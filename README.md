# 📲 수학 문제은행 PWA · 사용 안내

이 폴더에는 **수학 문제은행**을 윈도우/맥/안드로이드/아이폰에서 앱처럼 쓸 수 있는 PWA 파일들이 들어있습니다.

## 📁 파일 구성

```
math-bank-pwa/
 ├─ index.html          ← 메인 앱 (수정 가능)
 ├─ manifest.json       ← 앱 정보 (이름, 색상, 아이콘)
 ├─ service-worker.js   ← 오프라인 캐싱
 ├─ icon-192.svg        ← 일반 아이콘
 ├─ icon-512.svg        ← 고해상도 아이콘
 ├─ icon-maskable.svg   ← 안드로이드용 아이콘
 └─ README.md           ← 이 문서
```

## 🚀 사용 방법 — 3단계

### 1단계. 웹에 올리기 (HTTPS 필수)

PWA는 보안상 **HTTPS 환경**에서만 작동합니다. 가장 쉬운 무료 호스팅 방법:

#### 방법 A. **GitHub Pages** (추천 · 무료 · 영구)
1. github.com 가입 → 새 저장소(repository) 생성 (예: `math-bank`)
2. 이 폴더의 모든 파일을 저장소에 업로드 (드래그&드롭으로 가능)
3. 저장소 → `Settings` → `Pages` → Source를 `main` 브랜치로 설정
4. 1~2분 후 `https://[아이디].github.io/math-bank/` 주소가 생성됨

#### 방법 B. **Netlify Drop** (가장 간단 · 1분 소요)
1. https://app.netlify.com/drop 접속
2. 이 `math-bank-pwa` 폴더를 통째로 드래그&드롭
3. 즉시 `https://랜덤이름.netlify.app` 주소가 생성됨

#### 방법 C. **Vercel** / **Cloudflare Pages**
- 둘 다 무료, GitHub 저장소와 연동하면 자동 배포

---

### 2단계. 앱 설치

배포된 주소에 접속하면 **"📲 앱으로 설치하기"** 버튼이 사이드바 아래에 자동으로 나타납니다.

#### 윈도우 (크롬/엣지)
- 주소창 오른쪽의 **설치 아이콘(⊕)** 클릭, 또는 사이드바의 설치 버튼 클릭
- 시작 메뉴, 작업 표시줄, 바탕화면에 아이콘이 추가됨
- 일반 프로그램처럼 단독 창으로 실행됨

#### 안드로이드 (크롬)
- 사이드바의 **앱으로 설치하기** 버튼 클릭
- 또는 크롬 메뉴 → "홈 화면에 추가"
- 홈 화면에 아이콘 생성, 알림창 없이 앱처럼 실행

#### 아이폰/아이패드 (Safari 필수)
- iOS는 자동 설치를 지원하지 않으므로, 수동 설치:
  1. **Safari**로 사이트 접속 (크롬 X)
  2. 하단 **공유 버튼(□↑)** 누르기
  3. **"홈 화면에 추가"** 선택
  4. 홈 화면에 아이콘 생성됨

#### 맥 (크롬/엣지/Safari)
- 주소창의 설치 아이콘 클릭
- Launchpad에 등록되어 일반 앱처럼 실행

---

### 3단계. 사용하기

설치 후에는:
- ✅ **인터넷 없이도 실행 가능** (한 번 접속해서 캐시되면 OK)
- ✅ 단독 창으로 실행되어 브라우저 UI가 보이지 않음
- ✅ 윈도우 작업 표시줄, 안드로이드 멀티태스킹에 별도 앱으로 표시
- ✅ 아이콘, 스플래시 화면이 자동 적용

---

## 🔄 업데이트 방법

문제를 추가하거나 디자인을 바꾸려면:
1. `index.html` 파일을 텍스트 편집기로 열어 수정
2. 호스팅한 곳에 다시 업로드 (GitHub은 push만 하면 자동)
3. `service-worker.js` 안의 `CACHE_NAME` 버전을 올림 (`v1` → `v2`)
   → 사용자들이 다음 접속 시 자동으로 새 버전을 받음

---

## ⚠️ 주의사항

- **로컬 파일(file://)로 열면 PWA가 작동하지 않습니다.** 반드시 HTTPS 웹 주소로 접속해야 함
- 같은 PC/폰에 한 번만 설치 가능 (다시 설치 버튼은 안 보임)
- 설치 후 제거하려면: 윈도우는 `설정 → 앱`, 안드로이드는 길게 눌러 제거
